What is this?

After Flashing

‚úÖ Success! What's next?
  1. Disconnect and reconnect the USB cable
  2. Look for WiFi network: DataTracker-XXXX
  3. Connect to it with your phone or computer
  4. Browser should auto-open setup page (or go to 192.168.4.1)
  5. Configure WiFi and select your metric
  6. Save and enjoy your data tracker!

Troubleshooting

‚ùì Port not showing up
  • Try a different USB cable (must be data cable, not charge-only)
  • Try a different USB port on your computer
  • Make sure device is powered on
  • On Windows, install CH340 or CP2102 drivers if needed
‚ùì Flash failed / Connection error
  • Close any other programs using the serial port (Arduino IDE, PlatformIO, etc.)
  • Try holding the BOOT button while connecting USB
  • Try the "Factory Reset & Install" option instead
  • Restart your browser and try again
‚ùì Device not creating WiFi network after flash
  • Wait 30 seconds after flashing before disconnecting
  • Disconnect and reconnect USB to restart device
  • Try the "Factory Reset & Install" to clear old config
  • Check display - should show "SETUP MODE" instructions
