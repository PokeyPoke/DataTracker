### Flash Process
1. Load `flash.html` from GitHub Pages
2. Download binaries (bootloader, partitions, boot_app0, firmware)
3. User grants serial port access
4. Enter bootloader mode via DTR/RTS
5. Erase flash (optional)
6. Write bootloader at 0x0
7. Write partitions at 0x8000
8. Write boot_app0 at 0xe000
9. Write firmware at 0x10000
10. Reset device
11. Success!
